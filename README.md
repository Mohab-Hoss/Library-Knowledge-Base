# Librarian KR Chatbot — Streamlit

Run a KR-powered librarian tool with a web UI.

## Quick start
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## What it shows
- Taxonomy + FOL closure (ancestors & conflicts)
- Frames (Book slots)
- Production rules (explainable classification)
- Naïve Bayes backup (bn.json)
- Search + category filter (classification-on-the-fly)
